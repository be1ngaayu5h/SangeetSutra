# SangeetSutra

SangeetSutra is a project aimed at creating a seamless and enjoyable experience for music enthusiasts to explore and discover diverse genres of music.
This web application combines HTML, CSS, and JavaScript to provide users with a user-friendly interface for browsing, searching, and listening to music.

## Technologies Used

- HTML5
- CSS3 
- JavaScript 

## Usage

To run the project locally:
1. Clone this repository.
2. Open `index.html` in your web browser.
